# Fotografias de Profissionais do Direito

Esta pasta contém as imagens dos profissionais que aparecem na interface da EJUP.

## Imagens atualizadas

2. `person2.jpg` - Substitua pelo retrato do homem com tablet (segunda imagem fornecida)

## Recomendações

- As imagens já estão em formato apropriado para uso profissional
- Mantenha as proporções originais das imagens
- Dimensões recomendadas: 200x200 pixels (recortadas em formato quadrado)
- As imagens serão exibidas em formato circular na interface

Estas fotografias serão utilizadas na seção "Hero" da página inicial, mostrando os profissionais capacitados pela EJUP.
